# React + Tailwind 포트폴리오

이 프로젝트는 Vite, React, Tailwind CSS를 사용한 다크 모드 포트폴리오 템플릿입니다.

## 사용 방법

1. Node.js와 npm을 설치합니다. (공식 사이트 참고)
2. 프로젝트 루트(`my-vide-app`)에서 의존성을 설치합니다.

```bash
npm install
```

3. 개발 서버를 실행합니다.

```bash
npm run dev
```

4. 브라우저에서 `http://localhost:5173`에 접속합니다.

## 커스터마이징

- `src/App.jsx`에서 이름, 소개 문구, 프로젝트 목록, 연락처 정보를 수정하세요.
- Tailwind 관련 설정은 `tailwind.config.js`, 공통 스타일은 `src/index.css`에서 변경할 수 있습니다.
